# yearring-app

> YearRing Fund Protocol — dApp frontend
> **Live:** [app.yearringfund.com](https://app.yearringfund.com)

The official web interface for YearRing Fund Protocol on Base Mainnet.
Supports deposit, redeem, lock, beneficiary management, and role-based admin panels.

## Stack

- Framework: React + Vite + TypeScript
- Wallet: wagmi v2 + viem
- Network: Base Mainnet (Chain ID 8453)

## Development

```bash
npm install
npm run dev
```

## Deployment

GitHub Pages via `docs/` directory. Push to `main` to deploy.
